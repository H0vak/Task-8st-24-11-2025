Integer Reversal Algorithm (C++)
Цей проєкт реалізує математичний алгоритм для розвороту цифр натурального числа у зворотному порядку без використання рядкових перетворень.

Технології:
C++.

Тип даних long long — обраний для забезпечення стабільної роботи з великими числами та запобігання переповненню при розвороті.

Бібліотека <windows.h> — використана для коректної підтримки кириличного тексту в консолі.

Що робить ця програма:
Математична декомпозиція: За допомогою оператора % 10 програма послідовно виділяє останню цифру числа.

Динамічна збірка: Використовує формулу reversed * 10 + n % 10 для поступового формування нового числа у зворотному порядку.

Оптимізація циклу: Цикл while ефективно зменшує розрядність вхідного числа до нуля, забезпечуючи завершення роботи алгоритму.

Валідація вводу: Орієнтований на роботу з натуральними числами, що мають два і більше знаків.
